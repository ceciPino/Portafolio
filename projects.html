{Plantasia}

Plantasia es una red social, pensada para reunir a toda la comunidad amante de las plantas que tengan la intención de compartir consejos, preguntas, datos, o sólo mostrar y comentar el estado de sus plantas. Fue desarrollada Mobile First y como Single Page Application. Tanto para autenticar al usuario como para crear y manejar datos, trabajamos con Firebase. regístrate aquí ;)

{Ghiblers}

Este proyecto originalmente llamado "Data Lovers" fue asignado para filtrar y ordenar data. Trabajamos con arreglos y objetos y usamos principalmente los métodos "filter" y "map". Además en los estilos trabajamos con Flexbox y Grid.
